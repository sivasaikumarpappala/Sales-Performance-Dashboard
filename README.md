# Sales-Performance-Dashboard

Project Overview:
•	This project analyzes sales and profit trends across regions, categories, and customer segments using Power BI. 
•	It includes KPIs such as Year Sales, Year Profit, Profit Margin, and Average Order Value, with comparisons against the previous year.
•	Implemented dynamic Top/Bottom N product selection using slicers, enabling users to compare performance of best and worst selling products based on sales or profit.
•	The dashboard helps business users make data-driven decisions by exploring KPIs and visual analytics.

Tools & Technologies:
•	Power BI
•	DAX (Data Analysis Expressions)
•	MySQL
•	MS Excel

Dataset Summary:
•	Dataset Name: Superstore Sales Dataset
•	Records: ~10,000
•	Type: Public retail dataset
•	Includes: Customer details, order information, regional data, product categories, sales, profit, discount, shipping details, etc.

Project Workflow:
• Cleaned and prepared the sales dataset using MS Excel by removing duplicates, correcting data types, and deriving necessary fields
• Loaded the cleaned dataset into a MySQL database and performed basic SQL calculations such as profit margin
• Connected Power BI to the MySQL database for automated data retrieval and data modeling
• Built a relational data model in Power BI and established relationships between tables
• Created DAX measures for KPIs including Year Sales, Year Profit, Profit Margin, and Average Order Value with comparison against the previous year
• Designed interactive visuals such as line charts, bar charts, donut charts, and KPI cards to track business performance across regions and product categories
• Implemented dynamic Top/Bottom N product analysis using slicers, enabling users to evaluate best and worst performing products based on sales and profit
• Developed an interactive navigation page to switch between the overview dashboard and product ranking screen
• Published the dashboard layout to provide business stakeholders with actionable insights and reduce manual reporting efforts
